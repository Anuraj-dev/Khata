# Screenshots

Drop your real screenshots into this folder using the **exact filenames below**, then
swap the placeholder `src` in the root [`README.md`](../../README.md) (each `<img>` has
a comment with its intended local path, e.g. `<!-- docs/screenshots/web-expenses.png -->`).

Recommended size: **portrait phone frame ~ 390×844** (or any consistent ratio). PNG or JPG.

| Filename | What to capture |
|---|---|
| `web-expenses.png`  | Web app — Expenses home (list + summary bar) |
| `web-insights.png`  | Web app — Insights page (charts) |
| `web-udhaar.png`    | Web app — Udhaar / person ledger |
| `pwa-install.png`   | Installed PWA (standalone window / "Add to Home Screen") |
| `mobile-home.png`   | Mobile (Capacitor/Expo) — home screen |
| `mobile-sms.png`    | Mobile — auto-logged UPI SMS expense |

Tip: capture web screenshots at a mobile viewport (Chrome DevTools device toolbar) so
they match the mobile frames visually.
